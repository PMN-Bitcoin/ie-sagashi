1. Overview

"Ie-Sagashi Assistant" (the "Extension") is committed to protecting user privacy. This extension does not collect, store, sell, or share any personal information.

2. Data Stored Locally

The only data stored by this extension is kept locally on your device:

  • Search preferences set by the user (budget, area, room type, etc.)
  • Property listings saved by the user (rent, layout, URL, etc.)
  • Claude API key optionally entered by the user

All data is stored exclusively in your browser (chrome.storage.local) and is never transmitted to the developer or any third party.

3. External Communications

The extension communicates with the following external services only:

  • Real estate sites (SUUMO, HOMES, CHINTAI, APAMAN): The extension reads property data from pages the user is already visiting. No data is sent to these sites by the extension.
  • Anthropic Claude API (optional): Only when the user provides their own API key, property details are sent to Anthropic's servers for AI analysis. No personally identifiable information is included. Communication occurs directly from your browser to Anthropic's servers.

4. Data We Do NOT Collect

  • Name, email address, phone number, or any personal identifiers
  • Browsing history or search history
  • Location data
  • Payment or financial information
  • Any other personally identifiable information

5. Remote Code

This extension does not use any remote code. All JavaScript, HTML, and CSS are fully contained within the extension package.

6. Data Deletion

Uninstalling the extension from your browser will delete all locally stored data. You may also reset your data from the extension's settings page.

7. Policy Updates

If this policy is updated, this page will be revised. Significant changes will be communicated through extension update notes.

8. Contact

For privacy-related inquiries, please contact us via the Chrome Web Store support page.
